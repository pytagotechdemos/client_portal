# HOW TO USE

## 1. LOGIN ADMIN
1. Buka: https://07-client-portal.vercel.app/login
2. Email: `admin@pytagotech.com`
3. Password: `admin123`

---

## 2. BUAT PROJECT BARU
1. Klik **Projects** → **+ New Project**
2. Isi form (nama project, client info)
3. Client otomatis dapat account & email

---

## 3. TAMBAH DELIVERABLE
1. Buka project → Tab **Deliverables**
2. Di form bawah, isi nama & type
3. Klik **Add**

---

## 4. UPLOAD HASIL KERJA
1. Klik nama deliverable
2. Upload file atau masukkan link (Figma, dll)
3. Klik **Upload & Request Review**
4. Status jadi "REVIEW"

---

## 5. CLIENT REVIEW
1. Client buka link portal dari email
2. Lihat **"Awaiting Your Review"**
3. Klik **Review Now**
4. Pilih: **Approve** / **Revision Request**

---

## 6. BUAT INVOICE
(Syarat: minimal 1 deliverable berstatus APPROVED)

1. Klik **Generate Invoice** di project
2. Set harga per deliverable
3. Klik **Generate Invoice**
4. Invoice terkirim ke client via email

---

## 7. CLIENT BAYAR
1. Client buka portal → Tab **Invoices**
2. Klik **View Invoice**
3. Klik **Pay with Duitku**
4. Selesai → Status auto jadi PAID

---

## 8. BRIEFS & CHANGE REQUESTS
- **Briefs**: Tab **Briefs** → Upload file/referensi
- **Change Requests**: Tab **Change Requests** → Request kerjaan tambahan

---

## 9. DISCUSSION / COMMENTS
- Tab **Discussion** untuk chat antara admin & client
- Real-time update
