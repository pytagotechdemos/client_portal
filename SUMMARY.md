# CLIENT PORTAL

## Apa Ini?
Portal untuk agency manage project dengan client.

## Tech
Next.js 14 • PostgreSQL • NextAuth • Tailwind CSS • Duitku Payment

## Roles
- **Admin**: Login di `/login`
- **Client**: Login via link `/portal/[token]`

## Features
- Project management
- Deliverable upload & review
- Invoice & payment
- Real-time comments
- Email notifications
